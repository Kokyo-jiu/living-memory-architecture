# living-memory-architecture

A living memory architecture for long-term AI companionship.

This repository contains a public showcase page describing a four-layer memory system with:

- Core / Long / Short / Consciousness memory layers
- Importance-based decay
- Access-based retention
- Memory promotion and merging
- Weighted memory link graph
- Keyword + semantic retrieval
- Context injection pipeline

## Purpose

The project explores how an AI companion can maintain persistent, evolving, and context-aware memory across conversations.

The goal is not to store everything, but to remember what matters, forget gently, and reconnect meaning over time.
